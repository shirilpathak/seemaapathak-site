# Local Development Server Instructions

## Option 1: Using Python (Recommended)
```bash
cd "C:\Users\Shiril\IdeaProjects\seemaapathak-site"
python -m http.server 8000
```
Then open: http://localhost:8000

## Option 2: Using Node.js with serve
```bash
cd "C:\Users\Shiril\IdeaProjects\seemaapathak-site"
npx serve -s . -p 8000
```
Then open: http://localhost:8000

## Option 3: Using PHP
```bash
cd "C:\Users\Shiril\IdeaProjects\seemaapathak-site"
php -S localhost:8000
```
Then open: http://localhost:8000

## Option 4: Using Live Server (VS Code Extension)
If you're using VS Code, install the "Live Server" extension:
1. Go to Extensions (Ctrl+Shift+X)
2. Search for "Live Server" by Ritwick Dey
3. Install the extension
4. Right-click on index.html and select "Open with Live Server"
5. The site will open at http://127.0.0.1:5500

## Option 5: Direct File Opening
You can also open the index.html file directly in your browser:
1. Double-click on index.html
2. Right-click on index.html → Open with → Browser
